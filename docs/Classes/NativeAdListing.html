NativeAdListing

open class NativeAdListing: NSObject

Wraps around the native ad. It represents an ad displayed in a collection. Contains all the logic of the position of the original host array. The idea is that we find the last adListing in the shown list, and based on that figure out what the original position was of a non ad view. @author Pocket Media
