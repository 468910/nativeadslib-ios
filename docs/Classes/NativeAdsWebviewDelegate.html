NativeAdsWebviewDelegate

open class NativeAdsWebviewDelegate: NSObject, UIWebViewDelegate

Controls an instance of a webview. Adds a load indicator to give feedback back to the user. It follows each redirect until it reaches the app store or website. Then we’ll call the delegate
