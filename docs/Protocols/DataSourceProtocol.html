  • dequeueReusableCell(identifier:indexPath:)

    From: DataSourceProtocol Return the cell of a identifier. - parameter: adType The type of ad which holds the reuse identifier for the specified cell. This parameter must not be nil. - parameter: indexPath The index path specifying the location of the cell. The data source receives this information when it is asked for the cell and should just pass it along. This method uses the index path to perform additional configuration based on the cell’s position in the collection view.
